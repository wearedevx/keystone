## ks source

Prints all environment variables

### Synopsis

Prints all environment variables.

Environment variables values can then be loaded using eval, for example.

Example: 
  $ ks source
  KEY=value
  OTHER_KEY=other_value
  
  $ eval "$(ks source)"
  $ echo $KEY
  value
  $ echo $OTHER_KEY
  other_value
